# Linux-minimal && myshell 
### How to use? 
```bash
make && make run-nographic 
```
> This will compile the project and launch QEMU for simulation. 
> For more details, you can refer to the `Makefile`. 

### How to close? 
- `Ctrl A` + `X` (simulated with QEMU)
